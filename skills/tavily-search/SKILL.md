# Tavily Search Skill

Command-line interface to the Tavily API for AI-optimized web search and content extraction.

## Environment Setup

Requires `TAVILY_API_KEY` environment variable. Get your API key at https://app.tavily.com

```bash
export TAVILY_API_KEY="your-api-key-here"
```

## Tool API

### search
Search the web with AI-optimized results.

- **Parameters:**
  - `query` (string, required): Search query
  - `-n` (integer, optional): Number of results (default: 5, max: 20)
  - `--deep` (flag): Use advanced search depth (default: basic)
  - `--topic` (string, optional): `general` (default) or `news`
  - `--days` (integer, optional): For news topic, limit to last N days (default: 7)

**Usage:**
```bash
node skills/tavily-search/scripts/search.mjs "latest AI developments"
node skills/tavily-search/scripts/search.mjs "climate change" -n 10
node skills/tavily-search/scripts/search.mjs "tech news" --topic news --days 3
node skills/tavily-search/scripts/search.mjs "quantum computing" --deep
```

**Output:**
- AI-generated answer (if available)
- Ranked sources with title, URL, content snippet, and relevance score

### extract
Extract clean content from URLs.

- **Parameters:**
  - `urls` (strings, required): One or more URLs to extract content from

**Usage:**
```bash
node skills/tavily-search/scripts/extract.mjs "https://example.com/article"
node skills/tavily-search/scripts/extract.mjs "https://site1.com" "https://site2.com"
```

**Output:**
- Extracted text content from each URL
- List of any failed URLs with error messages

## Features

- **AI-Optimized:** Returns answers generated by AI, not just links
- **Relevance Scoring:** Each result includes a relevance score
- **News Search:** Filter by news topic with date range
- **Content Extraction:** Clean text extraction from web pages
- **Rate Limited:** Respects API rate limits (check your plan)

## API Limits

- **Free Tier:** 1,000 searches/month
- **Starter:** 10,000 searches/month
- **Pro:** 100,000+ searches/month

See https://tavily.com/pricing for current limits.

## Error Handling

- Returns error if `TAVILY_API_KEY` is missing
- Returns HTTP error if API request fails
- Validates input parameters before making API calls
